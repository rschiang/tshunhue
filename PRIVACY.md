# Privacy Policy

Tshunhue does not require an account and does not collect analytics, advertising identifiers, telemetry, contacts, or typed text.

The app downloads catalogs and images directly from the HTTPS URLs selected by the user. Those independent hosts may receive ordinary request information, such as an IP address and user agent. Tshunhue stores settings, catalog data, cached images, and recent items locally on the device.

The optional iOS keyboard uses selected text or the current line only to search local catalogs. The query remains in memory while the keyboard is active and is not saved or sent to Tshunhue. With Full Access, the keyboard may download an image and copy it to the system pasteboard.

Third-party catalog and image hosts have their own privacy policies. Questions and content reports can be submitted through the project’s GitHub repository.
